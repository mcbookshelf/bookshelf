## üì¶ [Bookshelf - 3.0.0](https://github.com/mcbookshelf/bookshelf/releases/tag/v3.0.0)

> **üöÇ The Pipeline Update!**
>
> *This release overhauls the project structure with Beet, focusing on modularity and isolation of each module. It also introduces a new version loader that resolves conflicts between multiple versions.*
>
> **‚ö†Ô∏è BREAKING CHANGES**: This new version has a few breaking changes!

Bookshelf is now based on **Minecraft 1.21.5**.
Each module of Bookshelf is now shipped as a separate datapack for people wanting to use only a single module.
Introduced bundles to better group modules, replacing **Bookshelf** and **Bookshelf Dev** datapacks: **Bookshelf Runtime**, **Bookshelf Dev**, **Bookshelf Prefabs**, and **Bookshelf Suite**.

- <abbr title="Breaking Changes">‚ö†Ô∏è</abbr> **[#281](https://github.com/mcbookshelf/bookshelf/issues/281)** - Merged the `bs.biome` and `bs.weather` modules into a single `bs.environment` module.

- <abbr title="New Modules">üéâ</abbr> **[#272](https://github.com/mcbookshelf/bookshelf/issues/272)** - Added an automatically generated `bs.load` module, removing the need for manual `load` and `unload` tags. Bookshelf now also uses **LanternLoad**.
- <abbr title="Enhancements">‚ö°</abbr> **[#285](https://github.com/mcbookshelf/bookshelf/pull/285)** - Removed the need for `__help__` functions and `help` tags. These are now generated.
- <abbr title="Documentation">üìù</abbr> **[#285](https://github.com/mcbookshelf/bookshelf/pull/285)** - Migrated the documentation to **mcbookshelf.dev** and reworked the contribution guide to explain the new **Beet** pipeline.

Updated the metadata format, added new issue templates, and introduced workflows for automatic deployment to **Modrinth** and **Smithed**. Modules now also require a `README.md` and a `pack.png`.
- <abbr title="Download">üì•</abbr> **[#333](https://github.com/mcbookshelf/bookshelf/issues/333)** - Bookshelf now has its own [download manager](https://mcbookshelf.dev), allowing users to select modules along with their dependencies and create custom bundles.
- <abbr title="Download">üì•</abbr> **[#336](https://github.com/mcbookshelf/bookshelf/issues/336)** - Bookshelf is now available on **PyPI** for use with the **Beet** toolchain.
